USAGE IN DEMO:
-------------
The demo randomly selects from these traffic flows to simulate realistic network conditions.
Each flow is classified using port-based rules and then routed using AI-based path selection
that considers traffic requirements, network conditions, and path characteristics.

The AI controller makes intelligent routing decisions by:
1. Classifying traffic type based on destination port
2. Selecting optimal path based on QoS requirements
3. Monitoring performance metrics (latency, jitter, packet loss)
4. Providing optimization suggestions for load balancing

This dataset represents a realistic mix of modern network traffic types with appropriate
QoS requirements for each category.
